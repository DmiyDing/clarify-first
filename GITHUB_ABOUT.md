## 5. Two-sentence blurb（两句话介绍）

For README, Smithery listing, or dev.to. 用于 README、Smithery 页面或技术博客。

**[EN]**  
AI assistants often guess and proceed—which leads to wrong work and rework. Clarify First is an Agent Skill that adds a risk-based gate: when your request is ambiguous or high-impact, the agent pauses, summarizes what it understands, proposes 2–3 options, and waits for your confirmation before making changes or running commands.

**[中文]**  
AI 助手常会「猜着做」，导致做错和返工。Clarify First 是一个 Agent Skill，在模糊或高风险请求时让 agent 先暂停、梳理理解、给出 2–3 个选项，并在你确认后再改代码或执行命令。

## 6. Publishing to Smithery（发布到 Smithery）

**[EN]**  
1. Tag a release (e.g. `v0.1.0`).  
2. Go to [smithery.ai/new](https://smithery.ai/new), sign in, publish a **Skill** (not MCP server), enter repo URL and skill path (`skills/clarify-first`).  
3. On the skill page: add the install command and the before/after example from the README.

**[中文]**  
1. 打 release/tag（如 `v0.1.0`）。  
2. 打开 [smithery.ai/new](https://smithery.ai/new)，登录后选择发布 **Skill**（非 MCP Server），填写仓库 URL 与 skill 路径（如 `skills/clarify-first`）。  
3. 在 skill 页面写上安装命令并粘贴 README 中的前后对比示例。
